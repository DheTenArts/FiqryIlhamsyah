Hi, Saya Fiqry Ilhamsyah Hasibuan

Saya mahasiswa Software Engineering yang memiliki ketertarikan besar di dunia teknologi dan pengembangan perangkat lunak. Saya senang mempelajari hal-hal baru, membangun solusi digital, serta terus mengembangkan kemampuan melalui berbagai proyek dan pengalaman belajar.

Perjalanan saya di dunia digital dimulai dari keinginan untuk menjadi seorang developer. Pada awalnya, saya belum memiliki gambaran yang jelas ingin berkembang di bidang apa, namun rasa penasaran terhadap teknologi membuat saya terus belajar dan mencoba berbagai hal. Seiring waktu, saya menemukan ketertarikan yang kuat di dunia game development. Dunia game bagi saya adalah ruang di mana teknologi, logika, dan kreativitas bisa berpadu untuk menciptakan pengalaman yang hidup dan interaktif. Dari sinilah arah perjalanan saya sebagai developer mulai terbentuk.
